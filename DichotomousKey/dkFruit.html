열매 종류의 검색표

1. 열매는 1개의 꽃이 성숙하여 됨 -->2

1. 열매는 여러 개의 꽃이 모여서 하나를 이룸. ==>(     다화과     )-->3

2. 1개의 열매(심피 1개 또는 여러 개 그리고 융합됨) ==(     단과     )-->3

2. 여러 개의 구별되는 열매(심피 여러 개, 각각 구별됨) ==(     취과     )-->3

3. 열매는 익어도 열개되지 않는다(     폐과     )-->4

3. 열매는 익으면 열개되거나 찢어진다(     개과     ) --> 13

4. 열매는 다육질성(적어도 부분적으로) -->5

4. 열매는 건조(     건과     ) -->8

5. 열매의 조직은 어느 정도 동질성(종자부위 제외), 전반적으로 다육질성 ==>(     장과     )

5. 열매를 구성하는 조직 이질성 --> 6

6. 열매의 외과피는 견고하고, 딱딱하며 또는 혁질이다; 내과피는 부드러운 육질로 구성됨 ==>(     장과     )

6. 열매의 외과피는 부드럽다; 내과피는 지질, 연골질, 또는 단단하다 --> 7

7. 열매의 중앙부위에 1개 또는 그 이상의 종자가 단단한 부막조직의 구조로 싸여짐; 자방하위 또는 자방상위 ==>(     핵과     )

7. 열매의 중앙부위는 지질 또는 연골질의 구조가 종자를 감싸고 있음; 자방하위 ==> (     이과     )

8. 열매는 보통 여러개 또는 다수의 종자를 가짐 ==>(     폐협과     )

8. 열매는 보통 1개의 종자를 가짐 --> 9

9. 열매에 날개가 있음 ==>(     시과     )

9. 열매에 날개가 없음 --> 10

10. 과피는 두텁고 딱딱한 후막조직으로 된 목질이다; 크기 대체로 크다 ==>(     견과     )

10. 과피는 얇고, 막질이다; 크기는 다소 작다 -->11

11. 과피는 종자(종피)와 느슨하게 융합되어 있꺼나 분리되어 있다. ==>(     낭과     )

11. 과피는 종자(종피)와 견고하게 밀착하여 융합되어 있다. --> 12

12. 과피는 종자에 견고하게 밀착되어 있지만, 종자로부터는 분리됨 ==> (     수과     )

12. 과피는 종자와 이합되어 밀착되어 있음.==>(     영과     )

13. 열매는 1개의 심피로부터 구성된다. --> 14

13. 열매는 2개-여러 개의 자성기로부터 구성된다. --> 16

14. 열매는 1개의 봉선을 따라 열린다. ==> (     골돌과     )

14. 열매는 2개의 봉선을 따라 열리거나, 횡선에 따라 열린다. -->15

15. 2개의 봉선이 종적으로 열린다. ==> (     협과     )

15. 봉선은 횡적으로 1개의 종자를 가진, 여러 조각으로 나누어진다. ==>(     분리과     )

16. 열매는 초기부터 점차로 찢어지는 건조/섬유질부터 혁질 또는 다육질의 외과피를 가지며, 중앙부위는 종자를 단단한 후막조직이 감싼다 ==>(     열개핵과     )

16. 열매는 종자를 둘러싼 단단한 후막조직이 없으며, 찢어져 열리거나 1개의 종자를 지닌 조각으로 분리된다. --17

17. 열매는 1개/몇 개의 종자를 가진 조각을 나뉜다(분과) ==> (     분열과     )

18. 열매는 종자가 붙었던 가장자리 주변의 남아있는 부분으로부터 2개의 자방실과 2개의 막으로 나뉘어진다. ==>(     장각과     )

18. 열매는 1개부터 여러 개의 자방실로 나누어지며, 만일 2개의 자방실의 경우 얇은 격막은 영속적으로 남지 못한다. ==>(     삭과     ) --> 19

19. 열매는 포주열개 방식인 횡적으로 열리고, 끈 부분은 뚜껑처럼 열리게 된다. ==> (     횡열개 삭과     )

19. 열매는 포주열개의 방식으로 열리지 않는다. --> 20

20. 열매는 구멍, 덮게 모양, 치아모양으로 벌어진다. --> 21

20. 열매는 종적으로 또는 불규칙적으로 벌어진다. --> 22

21. 열매는 정단의 연속적인 치간모양으로 열린다. ==> (     치상개열 삭과     )

21. 열매는 끝이나 밑에 구멍이 터져 벌어진다. ==> (     포공개열 삭과     )

22. 열매는 불규칙적으로 벌어진다. ==> (     부정열개 삭과     )

22. 열매는 종적으로 갈라져 터진다. --> 23

23. 격벽(자방실 사이의 격리막)으로 부터 판막이 부서지면서 터짐 ==>(     포축개열 삭과     )

23. 격벽(적어도 일부분)에 붙은 판막이 남아있음 --> 24

24. 열매는 심피실 격벽을 따라 터짐 ==>(     포간개열 삭과     )

24. 열매는 자방실과 격벽사이, 즉 배봉선을 따라서 터짐 ==>(     포배개열 삭과     )
