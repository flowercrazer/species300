참나무과

속 검색표

1. 열매는 1개의 견과가 들어있는 도토리(acorn)이며, 견과는 횡단면이 둥글고, 인편이 있는 각두에 부분적으로 싸인다. 각두는 갈라지지 않고, 봉선이나 관열이 없으며, 인편은 가시 모양으로 발달하지 않는다 -- 4. Quercus(     참나무속     )

1. 열매는 1~여러 개의 견과이며, 견과는 횡단면이 삼각형이거나 둥글고, 성숙할 때까지 가시가 있는 각두에 완전히 밀폐된다. 총포편은 2~4 (~8 또는 이상)이며, 이생하거나 봉선을 따라 가장자리가 유합한다. -->2

2. 수꽃은 다수가 두상화서에 달리며, 암꽃은 같은 화경에 쌍으로 달린다. 총포편은 4장이고, 견과는 삼각상이며, 자엽은 발아시 지상부로 나온다 ==> 1. Faqus (     너도밤나무속     )

2. 수꽃은 곧게 서는 미상화서에 달리며, 암꽃은 총포당 1~3개씩 달린다. 총포는 갈라지지 않거나 불규칙하게 갈라지며, 견과는 구형이며, 자엽은 발아시 지하부에 남는다. -->3

3. 낙엽성 교목으로 정아는 없다. 암꽃은 총포당 1~3개씩 달리고, 자방은 6실이며, 화주는 6개, 이생한다 ==> Castanea(     밤나무속     )

3. 상록성 교목으로 정아가 있다. 꽃은 총포당 1개씩 달리고, 자방은 3실이며, 화주는 3개, 기부에서 합생한다 ==> Castanopsis (     모밀잣밤나무속     )

2. Castanea Mill. 밤나무속

종 검색표

1. 잎 하면에는 흔히 인편상 분비선이 없으며, 적어도 어릴 때는 면모나 성상모가 밀생한다 ==> mollissima (     약밤나무     )

1. 잎 하면에 인편상 분비선 혹은 면모가 있다 ==> creanata (     밤나무     )

3. Castanopsis (D. Don) Spach 모밀잣밤나무속

종 검색표

1. 소지는 가늘다. 엽연은 전연이거나 드물게 선단부에만 짧은 예거치가 있으며, 하면에는 적갈색 또는 은색 털이 있다. 견과는 난상 구형(길이/너비 = 0.8~1.3), 길이 0.6-1.3 cm이다 ==> cuspidata (     모밀잣밤나무     )

1. 소지는 굵다. 엽연에는 짧은 예거치가 있으며, 하면에는 갈색 털이 있다. 견과는 난형 또는 타원형(길이/너비 = 1.4~2.2), 길이 1.2~2.1 cm이다 ==> sieboldii (     구실잣밤나무     )

4. Quercus L. 참나무속

종 검색표

1. 낙엽성 교목이며, 각두의 포린은 복와상이다. -->2

2. 잎에는 가시같은 거치가 있다. 열매는 2년만에 익는다. -->3

3. 수피에 코르크층이 발달한다. 하면에는 백색 성상모가 밀생한다 ==> variabilis (     굴참나무     )

3. 수피에 코르크층이 발달하지 않는다. 하면은 맥액에 충생모를 제외하고는 무모이거나 단모가 드물게 있다 ==> acutissima (     상수리나무     )

2. 잎에는 가시같은 거치가 없다. 열매는 당해에 익는다. -->4

4. 엽저는 이저이고, 엽병은 길이 1 cm 이하이다. -->5

5. 가지는 굵고, 적어도 어릴 때는 황색의 장연 또는 밀던모(성상모)가 밀생한다. 엽연의 거치는 둥근 치아상이다. 각두의 포린은 선형이고, 신장되며, 넓게 퍼지거나 다소 젖혀진다 ==> dentata (     떡갈나무     )

5. 가지는 가늘고, 무모이거나 어릴 때는 약간의 단모가 있다. 엽연의 거치는 불규칙하고, 파상이며, 선단부는 무디거나 뾰족하다. 각두의 포린은 짧고 눌려있다 ==> monqolica (     신갈나무     )

4. 엽저는 설저이고, 엽병은 길이 1~3 cm이다. -->6

6. 잎은 길이 10~25 (~35) cm이고, 측맥은 10~15쌍이며, 하면에는 성상모만 있다 ==> aliena (     갈참나무     )

6. 잎은 길이 6~15 cm이고, 측맥은 7~12쌍이며, 하면에 단모와 성상모가 밀생한다 ==> serrata (     졸참나무     )

1. 상록성 교목이며, 각두는 동심원상이다. -->7

7. 잎은 전연이거나 선단부에 소수의 예거치가 있다 ==> acuta (     붉가시나무     )

7. 잎에는 최소한 5 이상의 거치가 있다. -->8

8. 하면에는 황갈색 털이 밀생한다. -->9

9. 잎은 도피침형 내지 넓은 도피침형이며, 성숙한 잎의 하면, 소지, 엽병에는 황갈색 털이 밀생한다 ==> gilva(     개가시나무     )

9. 잎은 장타원형, 타원형 내지 도란상 장타원형이며, 하면에 회갈색 털이 산생한다 ==> glauca (     종가시나무     )

8. 하면에는 털이 거의 없고, 회백색이다. -->10

10. 가지는 다소 굵고, 자갈색이다. 엽연에는 짧고 뾰족한 소예거치가 있으며(느티나무형), 하면에는 털이 거의 없고 뚜렷한 회백색이다. 견과는 당해 가을에 익는다 ==> myrsinifolia (     가시나무     )

10. 가지는 가늘고 회갈색이다. 엽연에는 길고 뾰족한 예거치가 있으며(졸참나무형), 하면에는 처음에는 털이 있었으나 모두 없어지고, 백남질이 된다. 견과는 이듬해 가을에 익는다 ==> salicina (     참가시나무     )
